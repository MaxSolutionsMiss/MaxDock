MaxDock v34 — Horizontal Dock Swimlanes

Version 33 remains unchanged as the baseline.

Upload all six v34 files:
- index.html
- dashboard.html
- settings.html
- admin.html
- maxdock.css
- maxdock.js

Changes from v33 only:
- Dock Schedule is now horizontal.
- Dock 1, Dock 2, and additional configured docks appear vertically as swimlanes.
- Time runs horizontally across the schedule.
- Timeline options:
  - 30-minute detail
  - Hourly
  - 2-hour overview
- Previous day, Today, and Next day controls were added.
- Two dashboard metrics were added:
  - Inbound Skids
  - Outbound Skids
- All existing v33 requester, settings, location themes, dashboard controls, appointment list, and CSV export remain.

The skid totals cover non-cancelled appointments for the selected date and location.
